# 🔄 Módulo 1: Control de Lotes (Ciclos)

En una farmacia, revisar el inventario a mano puede tomar horas. Los **ciclos** (como el `for`, `while` o `do-while`) nos permiten automatizar tareas repetitivas en nuestro software, como verificar la fecha de caducidad de cientos de frascos de medicina en una fracción de segundo.

<details>
  <summary><b>👀 Haz clic aquí para ver el Ejercicio Práctico y el Código</b></summary>
  <br>
  
  Para este ejercicio, usamos un bucle `for` para simular la revisión automática de un lote de 5 frascos de Paracetamol. Para hacerlo profesional, le agregamos interactividad visual usando la clase `JOptionPane` de la librería **Java Swing**, la cual muestra el reporte final en una ventana emergente en lugar de usar la consola tradicional.
