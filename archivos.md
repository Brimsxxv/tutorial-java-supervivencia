# 💾 Módulo 5: Generación de Facturas (Manejo de Archivos)

Cuando un cliente realiza una compra en la farmacia, la información no puede simplemente desaparecer al cerrar el programa; necesitamos un comprobante físico o digital para la contabilidad. Aquí es donde entra el **manejo de archivos**.

Java nos permite comunicarnos con el disco duro de la computadora utilizando clases como `FileWriter` para crear y escribir archivos de texto (como facturas o reportes de inventario).
